# Scripts

These are the final useful analysis scripts retained for the revised manuscript.

Recommended order:

```bash
Rscript scripts/run_final.R /path/to/project_root
Rscript scripts/run_final_part2.R /path/to/project_root
Rscript scripts/run_final_part3.R /path/to/project_root
Rscript scripts/polish_figures.R /path/to/project_root
```

Excluded older exploratory scripts include earlier non-final filtering, Mantel tests, Levins niche breadth, sensitivity rarefaction, and generalist/specialist drafts. Those analyses were not part of the final minimal revision strategy.
